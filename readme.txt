Drive engagement to your website with the Tic-Tac-Toe Game.

== Description ==

Wp Tic-Tac-Toe allows visitors to play the game on your website, thereby driving engagement. There are four defined levels; Beginner, Learner, Pro, and Geek. Players can choose to learn with the Beginner & Learner Levels or Play to Beat the AI in Pro and Geek.

The Game can be added to any page or post via an easily accessible shortcode, [wp-tic-tac-toe] or conveniently set up on a sidebar in Appearance >> Widgets. Add the plugin, and you are ready to play!

Major features in Wp Tic-Tac-Toe include:

* Increasing Levels of Difficulty.
* Shortcode for Page/Post Inclusion.  
* Low memory usage.
* Well designed Game Layout


 
== Installation ==
 
The game can only run once. Please choose to display either on sidebar or page. Two instances of the game will trigger an error. e.g Having two games on a sidebar or displaying on a page and sidebar.

 
1. Upload the plugin to your Blog.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Look for Wp Tic-Tac-Toe at Wp-Admin (mostly at the bottom).
4. Add the Widget to a Sidebar, Appearance >> Widgets or Copy the shortcode, [wp-tic-tac-toe], and add to relevant pages.
5. All set!
